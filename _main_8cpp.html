Детальний опис

Головна програма моделювання системи автоматичного регулювання авіаційного двигуна.

Реалізує консольний інтерфейс для введення параметрів системи, цикл чисельного інтегрування методом Рунге-Кутта 4-го порядку та експорт результатів у CSV-формат з автоматичною візуалізацією.

Порядок роботи програми:

  1. Введення параметрів системи (T, r, k1, k2, k3, F0, alpha)
  2. Введення параметрів симуляції (t_start, t_end, h)
  3. Чисельне інтегрування методом RK4
  4. Збереження результатів у simulation_results.csv
  5. Автоматичний запуск visualize.py для побудови графіків

◆ main()

int main ( )

Точка входу програми моделювання CAP авіаційного двигуна.

Виконує повний цикл моделювання:

  • зчитує параметри системи та симуляції від користувача;
  • запускає чисельне інтегрування методом RK4;
  • зберігає результати (t, x, x', x'', x''', x^(4), F) у CSV;
  • запускає Python-скрипт visualize.py для побудови графіків.
Повертає
0 при успішному завершенні, 1 при помилці відкриття файлу.

◆ readDoubleWithDefault()

double readDoubleWithDefault ( const std::string & prompt,
double defaultValue,
double minValue = -std::numeric_limits<double>::infinity(),
double maxValue = std::numeric_limits<double>::infinity() )

Зчитує дійсне число з введення користувача з підтримкою значення за замовчуванням.

Якщо користувач натискає Enter без введення значення — повертає defaultValue. Якщо введене значення виходить за межі [minValue, maxValue] — також повертає defaultValue. Якщо введення не є числом — повертає defaultValue.

Аргументи
promptРядок запиту, що виводиться користувачу
defaultValueЗначення за замовчуванням (при порожньому введенні)
minValueМінімально допустиме значення (за замовчуванням -inf)
maxValueМаксимально допустиме значення (за замовчуванням +inf)
Повертає
Введене користувачем значення або defaultValue
Нотатки
Функція не кидає виняток — при будь-якій помилці повертає defaultValue.
