◆ State

using State = std::array<double, SYSTEM_ORDER>

Вектор стану системи у поточний момент часу.

Вектор містить: {x, x', x'', x'''}, де:

  • x — кількість обертів валу турбіни (об/с)
  • x' — перша похідна (швидкість зміни обертів)
  • x'' — друга похідна (прискорення)
  • x''' — третя похідна
