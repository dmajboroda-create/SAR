Детальний опис

Клас для чисельного інтегрування методом Рунге-Кутта 4-го порядку.

Реалізує класичний RK4-алгоритм для розв'язання систем звичайних диференціальних рівнянь вигляду dy/dt = f(t, y).

Формула RK4:

k1 = f(t, y)
k2 = f(t + h/2, y + h*k1/2)
k3 = f(t + h/2, y + h*k2/2)
k4 = f(t + h, y + h*k3)
y_{n+1} = y_n + (h/6)*(k1 + 2*k2 + 2*k3 + k4)
