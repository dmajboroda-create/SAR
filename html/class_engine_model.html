Детальний опис

Клас математичної моделі CAP авіаційного двигуна.

Містить параметри системи, функцію збурення F(t) та її похідні, а також метод обчислення правої частини системи диференціальних рівнянь.

Усі члени класу є статичними, що відповідає концепції єдиної глобальної моделі двигуна в рамках одного сеансу моделювання.

Нотатки
Перед використанням необхідно встановити всі параметри: T, r, k1, k2, k3, F0, alpha.
Застереження
Параметр T не може дорівнювати нулю (сингулярна система).

◆ computeDerivatives()

State EngineModel::computeDerivatives ( double t,
const State & current_state )
inlinestatic

Обчислює вектор похідних стану системи dy/dt = f(t, y).

Реалізує зведення диференціального рівняння 4-го порядку до системи 4 рівнянь 1-го порядку:

\begin{eqnarray*}\dot{y}_0 &=& y_1 \quad (dx/dt = x') \\ \dot{y}_1 &=& y_2 \quad (dx'/dt = x'') \\ \dot{y}_2 &=& y_3 \quad (dx''/dt = x''') \\ \dot{y}_3 &=& x^{(4)} \quad \text{з рівняння (1)} \end{eqnarray*}

де x^(4) обчислюється з основного рівняння:

\[x^{(4)} = \frac{k_1 T F''' + (k_1 + rTk_2)F'' - (1 + rTk_2)x''' - Tk_1k_2k_3 x''}{T} \]

Аргументи
tПоточний момент часу (секунди)
current_stateПоточний вектор стану {x, x', x'', x'''}
Повертає
Вектор похідних {x', x'', x''', x^(4)}
Обробка виняткових ситуацій
std::runtime_errorякщо T == 0 (сингулярна система)

◆ F()

double EngineModel::F ( double t)
inlinestatic

Обчислює значення функції збурення F(t) у момент часу t.

Функція моделює експоненціально спадаючу подачу палива до двигуна:

\[ F(t) = F_0 \cdot e^{-\alpha t}, \quad t \geq 0 \]

Аргументи
tМомент часу (секунди). При t < 0 повертає 0.
Повертає
Значення збурення F(t) >= 0.
Нотатки
Функція завжди повертає невід'ємне значення.
