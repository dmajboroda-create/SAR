\doxysubsection{Detailed Description}
Головна програма моделювання системи автоматичного регулювання авіаційного двигуна. 

Реалізує консольний інтерфейс для введення параметрів системи, цикл чисельного інтегрування методом Рунге-\/Кутта 4-\/го порядку та експорт результатів у CSV-\/формат з автоматичною візуалізацією.\hypertarget{_main_8cpp_autotoc_md0}{}\doxysubsubsection{\texorpdfstring{Порядок роботи програми:\+}{Порядок роботи програми:\+}}\label{_main_8cpp_autotoc_md0}

\begin{DoxyEnumerate}
\item Введення параметрів системи (T, r, k1, k2, k3, F0, alpha)
\item Введення параметрів симуляції (t\+\_\+start, t\+\_\+end, h)
\item Чисельне інтегрування методом RK4
\item Збереження результатів у simulation\+\_\+results.\+csv
\item Автоматичний запуск visualize.\+py для побудови графіків
\end{DoxyEnumerate}

Зчитує дійсне число з введення користувача з підтримкою значення за замовчуванням.
