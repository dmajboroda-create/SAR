\doxysubsection{Detailed Description}
Клас для чисельного інтегрування методом Рунге-\/Кутта 4-\/го порядку. 

Реалізує класичний RK4-\/алгоритм для розв\textquotesingle{}язання систем звичайних диференціальних рівнянь вигляду dy/\+dt = f(t, y).

Формула RK4:\+ 
\begin{DoxyCode}{0}
\DoxyCodeLine{k1\ =\ f(t,\ y)}
\DoxyCodeLine{k2\ =\ f(t\ +\ h/2,\ y\ +\ h*k1/2)}
\DoxyCodeLine{k3\ =\ f(t\ +\ h/2,\ y\ +\ h*k2/2)}
\DoxyCodeLine{k4\ =\ f(t\ +\ h,\ y\ +\ h*k3)}
\DoxyCodeLine{y\_\{n+1\}\ =\ y\_n\ +\ (h/6)*(k1\ +\ 2*k2\ +\ 2*k3\ +\ k4)}

\end{DoxyCode}
